RFCs
----
All RFCs are available at http://www.ieft.org/rfc/rfcNNNN.txt, where NNNN is
an RFC number below.

959 FILE TRANSFER PROTOCOL (FTP). J. Postel, J. Reynolds. October 1985.

2228 FTP Security Extensions. M. Horowitz, S. Lunt. October 1997.

2246 The TLS Protocol Version 1.0. T. Dierks, C. Allen. January 1999.

3280 Internet X.509 Public Key Infrastructure Certificate and Certificate
     Revocation List (CRL) Profile. R. Housley, W. Polk, W. Ford, D. Solo.
     April 2002.
